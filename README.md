# Blendermorse
### Blender extension for automatic morse code keyframing

This project is still under construction, if you'd like to 
see what I have so far check the dev branch.

### Planned Functionality
A sidebar panel that allows you to set a plaintext message to
encode, a property to animate (such as emission strength), and 
basic parameters such as high/low values and time unit, then 
translates that plaintext into morse code and inserts keyframes
for the active object, so you don't have to painstakingly set 
every keyframe by hand.
